Getting Started

NGIBS isn't just a search engine - it's an intelligent browsing system that combines multiple AI models with advanced search capabilities to provide comprehensive, contextual answers.

What Makes NGIBS Different?
Unlike traditional search engines that return links, NGIBS processes your queries through advanced AI models and provides direct, intelligent answers with reasoning and context.

Quick Start

  1. Visit NGIBS: Go to huggingface.co/spaces/jaihodigital/ngibs-ai-search
  2. Choose Your Mode: Select from Quick Search Pro, Live Search, Deep Search, or Memory Chat
  3. Select AI Model: Pick the best AI model for your query type
  4. Ask Your Question: Type your query and get intelligent answers

Complete Step-by-Step Guide

1
Choose Your Search Mode First
Important: Always select your search mode before choosing an AI model. Each mode is optimized for different types of queries:
  • Quick Search Pro: Direct AI responses
  • Live Search: Web-augmented answers
  • Deep Search: Complex analysis (coming soon)
  • Memory Chat: Contextual conversations (coming soon)
2
Select the Right AI Model
Choose from the sidebar based on your needs:
  • Llama 3.3 70B: General-purpose, balanced responses
  • Gemini 2.0 Flash: Fast, experimental features
  • Sonoma Dusk/Sky: Advanced reasoning & creativity
  • DeepSeek V3: Technical & programming queries
  • Dolphin Mistral: Uncensored, unrestricted responses
3
Craft Your Query
Be specific and detailed. NGIBS works best with:
  • Clear, specific questions
  • Context about what you need
  • Preferred format (list, explanation, analysis)
  • Any constraints or requirements
4
Optional: Upload Files
Enhance your queries with context by uploading:
  • PDF documents for analysis
  • Text files for processing
  • Images for visual understanding
5
Review & Interact
Get comprehensive answers and:
  • Ask follow-up questions
  • Request clarifications
  • Save important conversations
  • Switch models for different perspectives

Features Explained in Detail

Quick Search Pro (Available)

Best for: Direct questions, factual queries, quick research, problem-solving

How it works: Connects directly to your selected AI model without web search, providing fast, focused answers based on the model's training data.

Example queries: "Explain quantum computing", "Write a Python function for sorting", "Analyze this business strategy"

Live Search (Available)

Best for: Current events, latest information, real-time data, trending topics

How it works: Combines AI intelligence with real-time web search, providing up-to-date information augmented with AI analysis.

Example queries: "Latest AI developments 2025", "Current stock market trends", "Recent climate change research"

Deep Search (Coming Soon)

Best for: Complex research, multi-step analysis, academic inquiries

How it will work: Multi-step reasoning process that breaks down complex queries, searches multiple sources, and synthesizes comprehensive answers.

Memory Chat (Coming Soon)

Best for: Long conversations, building on previous discussions, iterative problem-solving

How it will work: Maintains context across multiple interactions, remembering previous questions and building comprehensive understanding over time.

Tips & Best Practices

Query Optimization

Do:

  • Be specific and detailed
  • Provide context and background
  • Specify desired output format
  • Ask follow-up questions
  • Use appropriate AI model for task

Don't:

  • Ask overly vague questions
  • Expect real-time data without Live Search
  • Use Quick Search for current events
  • Ignore model selection importance
  • Forget to specify requirements
